# NoppaEditori
Kouluprojekti, tekstieditori ja noppa

Tämä projekti käyttää wxWidgets (https://www.wxwidgets.org) ja käytin
tätä projektia opetellakseni tekemään graafisen käyttöliittymän C++:lla.
